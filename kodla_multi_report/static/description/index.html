Multi Report

  1. Download and unzip folder to your Odoo addons folder.
  2. Go Settings -> Active Developer Mode
  3. Go Apps -> Update Apps List
  4. Search and find module in the list and install.
  5. Now you can print out you invoice in Invoice Page.
Important note!
If you can't see whatsapp button on your web page. Please try restart your odoo service.

How you can restart your odoo service?

Linux

sudo systemctl restart odoo17

Windows

  1. Press Win + R
  2. Type services and press Enter
  3. Find odoo-server-*
  4. Right click -> Restart
